Транслятор входных данных во входной язык решателя
Программа make_input.py, написанная на языке Python 3.5. Получает на вход файл с описанием графа работ и транслирует его в 5 файлов на входном языке решателя, соответствующих 5 различным упорядочениям вершин и ребер графа (а, следовательно, переменных и ограничений задачи ЦЛП).
Формат вызова программы:
make_input.py <файл входных данных>

Командный сценарий запуска серии экспериментов
Программа run_exp.sh, написанная на языке bash, получает на вход каталог с файлами входных данных (определяются по расширению «.txt»). Для каждого найденного файла:
- вызывает программу make_input.py для каждого найденного там файла с расширением «.txt», чтобы получить описание задачи ЦЛП с 5 вариантами упорядочения переменных и ограничений;
- запускает 5 экземпляров решателя параллельно (рекомендуется для использования на процессоре с не менее чем 6 ядрами), с лимитом времени 30 минут;
Формат вызова программы:
run_exp.sh <каталог с файлами входных данных>

Сборщик результатов экспериментов
Программа get_stats.py, написанная на языке Python 3.5. Получает на вход каталог с файлами результатов экспериментов и выводит в стандартный поток вывода два словаря с характеристиками графа и с данными о завершившихся экспериментах, соответственно. Данные содержат: статус завершения эксперимента, оптимальное значение целевой функции (если найдено), время выполнения эксперимента.
Формат вызова программы:
get_stats.py <каталог с файлами результатов экспериментов>

Программа отображения результатов экспериментов
Программа plot.py, написанная на языке Python 3.11 с использованием библиотеки matplotlib. Получает на вход два словаря с харкатеристиками графа и с данными о завершившихся экспериментах, соответственно, и строит диаграмму по серии экспериментов в формате, описанном в разделе 5.4.
Формат вызова программы:
plot.py <файл словаря1> <файл словаря2>

Папки 1_type, 2_type, 3_type содержат первый, второй, третий тип графов с известным оптимумом, соответственно.
